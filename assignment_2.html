Text Formatting Example

1.<b>

Use the <b> to make the text bold

Example : This text is bold.

2.<strong>

Use the <strong> to make the text bold

Example : This text is bold.

3.<em>

Use the <em> to make the text italicize

Example : This text is italicize.

4.<u>

Use the <u> to make the Underline under the text

Example : This text is underlined.

5.<sup>

Use the <sup> to make the text superscript

Example : x2+y2

6.<sub>

Use the <sub> to make text subscript

Example : Co2

7.<del>

Use the <del> to defines text that has been deleted

Example :My favorite color is blue green

8.<ins>

Use the <ins> to a text that has been inserted

Example :My favourtie color bluegreen

9.<mark>

Use the <mark> to defines a text that should be marked or highlighted

Example :Do not forget to buy milktoday.
